Embedding maps and videos.

Have you ever wondered how some websites are able to play videos from YouTube or other streaming platforms from the webpage itself. Or how you can have a section with a google maps with all its features still functioning. The answer to this is pretty simple, it’s thanks to the use of embedded code that web pages are to add different type of media. An embedded code is a block of HTML that is placed in another page and renders a visual element a video, social media post, form, or page from another website or source. You can embed YouTube videos, Twitter posts, GIFs, and other multimedia objects on another website.

What is embed.

Now you might be asking what is embed exactly. Well, embed in html means to integrate external content into another website or page. You embed something when you place a block of code called an embed code into the HTML editor of another website. When you hit ‘Save’ or ‘Publish,’ the media then renders on the published page.

How to Add HTML Embed Code.

Now when it comes to adding an HTML embed code, for each site the method of getting an html code will be slightly different depending on the site. In general, you would go to the social post or webpage you’d like to embed, generate the embed code using the post’s options this can either be the three dots or the share button. Next you will need to find and copy the html code then paste the HTML snippet you just copied into your HTML viewer window. Don’t forget to save it and with that you are now finished and successfully embedded content onto your website or blog.

What is iframe.

Now when you’re embedding a video or the map you will notice that it’s in the iframe element but just what is an iframe. An iframe or an inline frame (iframe) is a HTML element that loads another HTML page within the document. It essentially puts another webpage within the parent page. They are commonly used for advertisements, embedded videos, web analytics and interactive content. The iframe element is easy to use, well supported and allows the embedding of third-party content. However, it is a heavy processer and memory, you have no control of the third-party site and it’s difficult to use in responsive design pages.

How is iframe used?

The most common use of an iframe is to load content from another site within the page. The child site can load its own content and cookies, so sites may allow it where they don't allow direct hotlinking content. Using an iframe is the accepted way to embed a YouTube video or Google Maps content. The iframe works by creating a new HTML document environment to load the content within.

What is src?

Now when you’re embedding a video or the map you will also notice that there is a src attribute that’s then followed by a link. The src is an attribute that specifies the location (URL) of the external resource. It’s an important part of the embedding code.
